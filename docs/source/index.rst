=========
ChainerCV
=========


ChainerCV is a **deep learning based computer vision library** built on top of `Chainer <https://github.com/pfnet/chainer/>`_. 


Install Guide
-------------

Please install `Chainer <https://github.com/pfnet/chainer/>`_ first.
You can install ChainerCV using `pip`.

.. code-block:: shell

    pip install chainercv
